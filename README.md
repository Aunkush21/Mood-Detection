# 🧠 Mood Detection from Text (Machine Learning + Web App)

A full-stack Machine Learning project that predicts the **mood** behind any piece of text.  
The model classifies text into four moods:

- 😄 **Happy**
- 😢 **Sad**
- 😡 **Angry**
- 😐 **Neutral**

This project includes:

✔ A complete training pipeline in Python  
✔ Logistic Regression model with class balancing  
✔ Flask backend API  
✔ Responsive modern frontend (HTML + CSS + JS)  
✔ Basic UI + Emoji-based mood display  
✔ Fully functional end-to-end web app  

---

## 🚀 Features

### 🔍 Machine Learning
- Uses **Google's GoEmotions dataset**
- Maps 27 emotions into 4 mood categories
- Text vectorization using **TF-IDF**
- Class balancing with `class_weight="balanced"`
- 70%+ accuracy on test data
- Clean model saving and loading using Joblib

### 💻 Full-Stack Web Application
- Flask REST API endpoint: `/predict`
- Secure CORS-enabled backend
- Fully responsive UI
- Mood-based emoji display
- Works on all devices
